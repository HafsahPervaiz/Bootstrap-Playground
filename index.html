Fundamentals

Typography

Bootstrap gives you powerful utility classes to style text without writing custom CSS.

📖 Key Concepts
  • .display-1 to .display-6 — Large, eye-catching headings
  • .lead — Makes a paragraph stand out
  • .fw-bold, .fst-italic — Font weight & style
  • .text-center, .text-end — Text alignment
  • .text-uppercase, .text-lowercase — Transform text
  • .fs-1 to .fs-6 — Font size utilities

Bootstrap's color utilities let you style text and backgrounds with semantic color classes.

Grid System

Bootstrap's 12-column grid is the foundation of responsive layouts. Master it and you can build anything.

📖 Key Concepts
  • .container — Wraps content with responsive padding
  • .row — Creates a horizontal group of columns
  • .col-6 — Takes 6 out of 12 columns (50% width)
  • .col-md-4 — 4 columns on medium screens and up
  • .g-3 — Adds gap/gutter between columns
  • Breakpoints: sm, md, lg, xl, xxl

Flexbox Utilities

Bootstrap's flex utilities let you control layout alignment, direction, and spacing powerfully.

📖 Key Concepts
  • .d-flex — Enable flexbox on an element
  • .justify-content-center — Center items horizontally
  • .align-items-center — Center items vertically
  • .flex-column — Stack items vertically
  • .gap-3 — Add spacing between flex items
  • .flex-wrap — Allow items to wrap to next line
